О проекте

Курсовой проект

«Разработка интеграционной базы данных на основе парсинга веб-ресурсов»

Цель проекта

Целью проекта является приобретение практических навыков по сбору, обработке и интеграции разнородных данных из публичных веб-источников, проектированию реляционной базы данных и созданию простого веб-приложения для работы с этой базой.

Технологии

  • Backend: Python, Flask
  • База данных: SQLite
  • Парсинг: Requests, BeautifulSoup
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • API: RESTful API для доступа к данным
  • Визуализация: Chart.js для графиков
  • Дедупликация: Алгоритмы объединения дубликатов

Функциональность

Поиск товаров

По названию, бренду, цене

Сравнение цен

Из разных магазинов

Статистика

Анализ данных в реальном времени

Структура базы данных

Проект использует 4 основные таблицы:

  1. products - уникальные товары после дедупликации
  2. offers - предложения с конкретных сайтов
  3. attributes - характеристики товаров
  4. parse_logs - логи парсинга данных
Примечание

В данном проекте используются тестовые данные, сгенерированные автоматически. В реальной реализации данные парсятся с сайтов-источников.
